# Pakki-Pakki Gaming

Minimal bilingual teaser for **Retro Hockey Manager**, published by Pakki-Pakki Gaming.

## Local preview

Serve the parent directory so the preview uses the GitHub Pages subpath:

```sh
cd /Users/juhorautio
python3 -m http.server 8080
```

Open `http://localhost:8080/pakki-pakki-gaming/`.

## Publishing

GitHub Pages publishes the repository root from `main` to <https://spjass.github.io/pakki-pakki-gaming/>.

English and Finnish copy lives in `site.js`. The English teaser remains available when JavaScript is disabled. The site has no analytics, cookies, forms, authentication, or external runtime dependencies.
